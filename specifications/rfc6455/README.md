# WebSocket Protocol RFCs

This directory contains the WebSocket protocol specifications in markdown format for easy reading and reference.

## Main Specification

- **[rfc6455.md](rfc6455.md)** - The WebSocket Protocol (December 2011)
  - The core WebSocket protocol specification
  - Defines the opening handshake, data framing, and closing handshake
  - 148 KB, comprehensive specification

## Updates and Extensions

- **[rfc7936.md](rfc7936.md)** - Clarifying Registry Procedures for the WebSocket Subprotocol Name Registry (July 2016)
  - Updates RFC 6455
  - Clarifies IANA registry procedures for WebSocket subprotocols

- **[rfc8307.md](rfc8307.md)** - Well-Known URIs for the WebSocket Protocol (January 2018)
  - Updates RFC 6455
  - Defines well-known URI for WebSocket protocol metadata

- **[rfc8441.md](rfc8441.md)** - Bootstrapping WebSockets with HTTP/2 (September 2018)
  - Updates RFC 6455
  - Defines mechanism for running WebSocket over HTTP/2 connections

## Format Notes

- All RFCs converted from official RFC Editor text format
- Page headers and footers removed for clean, linear reading
- Section headers properly formatted with markdown heading levels
- ASCII art diagrams enclosed in code blocks
- RFC cross-references converted to clickable links
- No page numbers or formatting artifacts
